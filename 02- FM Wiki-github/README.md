## ⚙️ Instalação dos pacotes essenciais.

Instalar os pacotes:

```
pnpm install
```

Iniciar o Front-End:

```
pnpm run dev
```

## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

-   HTML e CSS.
-   JavaScript.
-   React.
-   Vite.
-   Styled Components.
-   Axios.
